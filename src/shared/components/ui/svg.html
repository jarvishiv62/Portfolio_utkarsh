◈ How to use in your portfolio

Every SVG uses currentColor for adaptive strokes — set color on the parent element and it follows your dark/light theme automatically.
The Indigo accent #6C3D91 is hardcoded so it stays vivid in both themes.
All CSS animations are scoped inside each SVG's own <style> tag — safe to paste as inline JSX with no conflicts.
For JSX: rename class= → className= and self-close tags where needed. Tip: wrap animated SVGs in a will-change: transform container for GPU acceleration.
